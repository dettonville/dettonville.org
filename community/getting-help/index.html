Getting Help

Edit on GitHub

You are using Dettonville and ran into a problem?

First check out the reference guide. If that doesn’t help to answer your question you may join the Dettonville GitHub Discussions or hop by the Dettonville Gitter room. We also monitor the dettonville tag on StackOverflow.

To report a bug or request a new feature use the Dettonville issue tracker. Note that bug reports should be accompanied by a test case demonstrating the issue.
